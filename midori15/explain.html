このドキュメントでは、Three.jsを使用して球体と水面エフェクトを作成する方法について説明します。

const geometry = new THREE.SphereGeometry(15, 32, 32);
        const material = new THREE.ShaderMaterial({
            uniforms: {
                time: { value: 0 },
                color: { value: new THREE.Color(0x000000) }
            },
            vertexShader: `...`,
            fragmentShader: `...`,
            transparent: true
        });
        const sphere = new THREE.Mesh(geometry, material);
        sphere.position.set(0, 0, 0);

THREE.SphereGeometryは球体の形状を定義します。ここでは、半径15、縦横32の分割で球体を作成しています。

THREE.ShaderMaterialを使用して、カスタムのシェーダーをマテリアルに適用しています。このシェーダーは、時間経過による波打つ効果を球体に適用します。

水面エフェクトの追加

次に、水面エフェクトをシーンに追加する方法を説明します。以下のコードは、水面エフェクトを作成し、それをシーンに追加する機能を持っています。

const waterGeometry = new THREE.PlaneGeometry(250, 250, 256, 256);
        const textureLoader = new THREE.TextureLoader();
        textureLoader.load('water_1.png', function(waterTexture) {
            waterTexture.wrapS = waterTexture.wrapT = THREE.RepeatWrapping;
            waterTexture.repeat.set(50 / waterTexture.image.width, 50 / waterTexture.image.height);
        
            const waterMaterial = new THREE.ShaderMaterial({
                uniforms: {
                    time: { value: 0 },
                    waterTexture: { value: waterTexture },
                    minHeight: { value: 0 },
                    maxHeight: { value: 1 }
                },
                vertexShader: `...`,
                fragmentShader: `...`,
                transparent: true
            });
        
            const waterMesh = new THREE.Mesh(waterGeometry, waterMaterial);
            waterMesh.rotation.x = -Math.PI / 2;
            scene.add(waterMesh);
        }, undefined, function(error) {
            console.error('テクスチャの読み込みに失敗しました:', error);
        });

THREE.PlaneGeometryは水面の形状を定義します。ここでは、250x250の平面を256x256の分割で作成しています。

テクスチャはTHREE.TextureLoaderを使用して読み込まれ、水のテクスチャとして適用されます。シェーダーはこのテクスチャを使用して、水面が時間とともに動くような視覚効果を生成します。
