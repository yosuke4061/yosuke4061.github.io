カスタム水エフェクトの追加

この関数は、Three.jsを使用して水の表面のエフェクトをシミュレートします。

コード解説

水のジオメトリの作成

const waterGeometry = new THREE.PlaneGeometry(50, 50, 256, 256);

50x50の平面を作成し、256x256の分割を行います。これにより、水の波紋がよりリアルに表現されます。

テクスチャの読み込み

const textureLoader = new THREE.TextureLoader();
textureLoader.load('_MG_0070.jpg', function(waterTexture) {...});

テクスチャローダーを使用して、水のテクスチャを読み込みます。読み込んだテクスチャは繰り返し設定を行い、シームレスな表現を可能にします。

シェーダーマテリアルの設定

const waterMaterial = new THREE.ShaderMaterial({...});

シェーダーマテリアルを使用し、カスタムの頂点シェーダーとフラグメントシェーダーを定義します。これにより、水の動きや高さの変化をリアルタイムで計算し、視覚的に表現します。

水のメッシュの作成とシーンへの追加

const waterMesh = new THREE.Mesh(waterGeometry, waterMaterial);
scene.add(waterMesh);

作成したジオメトリとマテリアルからメッシュを作成し、シーンに追加します。メッシュは水平面に配置されます。

アニメーションの実行

function animate() {
    requestAnimationFrame(animate);
    waterMaterial.    uniforms.time.value += 0.05;
    renderer.render(scene, camera);
}
animate();

アニメーション関数では、フレームごとに時間の値を更新し、シェーダーに渡して水の動きをシミュレートします。そして、シーンを再描画してアニメーションを継続します。

動作の概要

このコードは、水の表面が波打つエフェクトを生成します。テクスチャは水の見た目を提供し、シェーダーは波の動きを計算して、それをリアルタイムで画面に表示します。ユーザーは水の表面が時間とともに変化するのを見ることができます。

考え方

このエフェクトは、グラフィックスプログラミングにおけるシェーダーの利用を示す良い例です。頂点シェーダーで各頂点の位置を時間に応じて変化させ、フラグメントシェーダーでテクスチャに基づいてピクセルの表示を制御します。このようにして、複雑な自然現象を効率的にシミュレートすることが可能になります。
