createVolumetricCloud 関数の解説

概要

この関数は、Three.js を使用して体積雲を生成するためのものです。生成された雲は、シェーダーマテリアルを使用してリアルタイムで視覚的な効果が適用されます。

使用方法

const cloud = createVolumetricCloud(); scene.add(cloud);

コードの詳細

ジオメトリの作成

まず、THREE.SphereGeometry を使用して球体のジオメトリを作成します。このジオメトリは、x軸方向に非常に平たくスケールされ、楕円体を形成します。

const cloudGeometry = new THREE.SphereGeometry(5, 32, 32); cloudGeometry.scale(30, 0.01, 30);

シェーダーマテリアルの設定

シェーダーマテリアルは、頂点シェーダーとフラグメントシェーダーで構成されます。これにより、雲の見た目がリアルタイムで生成されます。

頂点シェーダーでは、頂点の位置を vPosition として渡し、フラグメントシェーダーで使用します。

フラグメントシェーダーでは、ノイズ関数を使用して雲の密度を計算し、その密度に基づいて色と透明度を設定します。

vertexShader: ` varying vec3 vPosition; void main() { vPosition = position; gl_Position = projectionMatrix * modelViewMatrix * vec4(position, 1.0); } `, fragmentShader: ` varying vec3 vPosition; uniform float time; uniform float amplitude; uniform vec3 baseColor; uniform vec3 viewPosition; uniform samplerCube envMap; ... `

マテリアルの設定

マテリアルには、時間、振幅、基本色、視点位置、環境マップなどのユニフォームが設定されています。これらはシェーダーで使用され、雲の動的な見た目に影響を与えます。

uniforms: { time: { value: 0.0 }, amplitude: { value: 5.0 }, baseColor: { value: new THREE.Color(0.8, 0.8, 0.9) }, viewPosition: { value: camera.position }, envMap: { value: 'cloud.jpg' } }

メッシュの作成と追加

最後に、ジオメトリとマテリアルを使用してメッシュを作成し、シーンに追加します。

const cloud = new THREE.Mesh(cloudGeometry, cloudMaterial); scene.add(cloud);
