Three.jsによる地形生成の解説

この記事では、Three.jsを使用して地形を生成し、それに霧の効果を加える方法について説明します。

addTerrain() 関数

addTerrain() 関数は、地形のメッシュを生成し、シーンに追加する役割を持っています。以下にその主要なステップを示します。

  1. 地形の幅と深さを定義します。
  2. 高さデータを生成するために generateHeight() 関数を呼び出します。
  3. Three.jsの PlaneGeometry を使用して地形のジオメトリを作成します。
  4. 生成した高さデータをもとに頂点の位置を調整します。
  5. テクスチャと法線マップを生成し、マテリアルに適用します。
  6. 最終的にメッシュを作成し、シーンに追加します。

コード例

function addTerrain() {
            const worldWidth = 256, worldDepth = 256;
            const data = generateHeight(worldWidth, worldDepth);
            const geometry = new THREE.PlaneGeometry(5000, 5000, worldWidth - 1, worldDepth - 1);
            geometry.rotateX(-Math.PI / 2);
            const vertices = geometry.attributes.position.array;
            for (let i = 0, j = 0, l = vertices.length; i < l; i++, j += 3) {
                vertices[j + 1] = data[i] * 10;
            }
            const texture = new THREE.CanvasTexture(generateTexture(data, worldWidth, worldDepth));
            const normalTexture = generateNormalMap(data, worldWidth, worldDepth);
            const material = new THREE.MeshStandardMaterial({
                map: texture,
                normalMap: normalTexture
            });
            const mesh = new THREE.Mesh(geometry, material);
            mesh.position.y = -500;
            scene.add(mesh);
        }

関連する関数

generateHeight()

generateHeight() 関数は、指定された幅と高さに基づいてランダムな高さデータを生成します。このデータは地形の頂点の高さを決定するために使用されます。

generateTexture()

generateTexture() 関数は、高さデータからテクスチャを生成します。このテクスチャは地形の表面に適用され、視覚的な詳細を提供します。

generateNormalMap()

generateNormalMap() 関数は、高さデータを使用して法線マップを生成します。法線マップを生成します。法線マップは、光の反射をリアルに再現するために使用され、地形のテクスチャに立体感を与えます。

霧の効果の調整

霧の効果は、THREE.Fog を使用してシーンに追加されます。霧は、遠くのオブジェクトをぼかすことで、よりリアルな景観を作り出すのに役立ちます。霧の密度は、fogNear と fogFar パラメータで調整できます。

コード例

scene.fog = new THREE.Fog(0xf0faff, 200, 1000);

この設定では、霧は200ユニットの距離で始まり、1000ユニットで完全に消失します。これにより、カメラから遠く離れた地点では霧によって視界が遮られ、近くの地形はクリアに見えるようになります。

まとめ

この記事では、Three.jsを使用して地形を生成し、霧の効果を加える方法について説明しました。これらの技術を駆使することで、リアルタイムで美しい3D環境を作成することが可能です。地形の解像度を高めることで詳細な地形を作り出し、霧の効果を適切に調整することで、視覚的な深みを加えることができます。
